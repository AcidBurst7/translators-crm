# Тестовое задание: доработка CRM для переводчиков

## 📌 Описание задания

Необходимо было доработать функционал CRM для работы с переводчиками:

- Переводчики работают в будние дни, но некоторые могут работать и в выходные.
- CRM должна позволять выбирать доступных переводчиков по дате и занятости.
- Нужно реализовать:
    1. Хранение переводчиков и их расписания в базе данных.
    2. API для получения списка переводчиков.
    3. Отображение списка переводчиков на фронтенде с помощью Vue.js.
    4. Сообщение, если переводчиков нет.

---

## 1. Установка зависимостей

```bash
docker-compose run --rm backend composer install
```

## 2. Инициализация приложения

```bash
docker-compose run --rm backend php /app/init
```

## 3 Запуск приложения

```bash
docker-compose up -d
```

## 5. Применение миграций
```bash
docker-compose run --rm backend yii migrate
```

## 6. Добавление тестовых данных (опционально)

```bash
php yii seed
```

## 7. API

Endpoint: /index.php?r=translator/list&date=YYYY-MM-DD

Пример ответа JSON:

```json
{
    "message": "Список переводчиков готов",
    "data": [
      { "id": 1, "name": "Иван Иванов" }
    ]
}
```

Если переводчиков нет:

```json
{
"message": "Нет свободных переводчиков",
"data": []
}
```

## 8. Доступ в браузере

Frontend: http://127.0.0.1:20080
Backend: http://127.0.0.1:21080
